Web Text Extractor 隐私政策

简介

本隐私政策说明了 Web Text Extractor（以下简称"我们"、"我们的"或"该扩展程序"）在您使用我们的浏览器扩展程序时如何收集、使用和保护您的数据。

数据收集

我们收集的内容

Web Text Extractor 仅收集：

  • 您主动选择提取的网页文本内容
  • 扩展程序核心功能所需的临时数据

我们不收集的内容

我们不收集：

  • 个人身份信息
  • 浏览器历史记录
  • 登录凭据
  • 财务信息
  • 位置数据
  • 任何其他个人或敏感信息

数据使用

我们如何使用您的数据

收集的数据仅用于：

  • 提供文本提取功能
  • 在扩展程序的侧边栏中显示选定的文本
  • 为方便您使用而临时存储提取的文本

数据存储

  • 所有数据都存储在您的浏览器本地
  • 不向外部服务器传输数据
  • 数据持久性仅限于您的浏览器的本地存储

数据保护

安全措施

  • 所有扩展程序功能都在您的浏览器本地运行
  • 无外部数据传输
  • 无远程代码执行
  • 定期进行安全更新和维护

第三方访问

我们不会：

  • 与第三方共享您的数据
  • 出售您的数据
  • 将您的数据用于广告目的

用户权利

您有权：

  • 控制要提取的文本内容
  • 随时删除本地存储的数据
  • 禁用或卸载扩展程序

隐私政策的变更

我们可能会不时更新本隐私政策。任何变更都将在本文档中反映，并附上更新后的版本。

联系方式

如果您对本隐私政策有任何疑问，请通过 Chrome Web Store 支持渠道与我们联系。
